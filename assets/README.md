# Assets Folder

Place your game assets in this folder.

## Background Music
To add background music to the game:
1. Prepare an MP3 file (recommended format for broad browser compatibility)
2. Name it `bgm.mp3`
3. Place it in this folder (`assets/bgm.mp3`)
4. Reload the game - the music will automatically play when you interact with the game

## Other Assets
You can also add other assets like:
- Sound effects (place in this folder and reference in main.js)
- Images for towers/enemies (would require modifying the drawing code)
- etc.

Note: For the best experience with cloud deployment, keep audio files reasonably sized (<1MB recommended).
