# 디펜스 게임

이 게임은 웹 기반 타워 디펜스 게임입니다. 적이 정해진 경로를 따라 오면 플레이어는 타워를 배치하여 그들을 막아야 합니다.

## 특징

- 다양한 타워 타입 (기본, 느림, 스플래시)
- 다양한 적 유형 (기본, 빠름, 탱크)
- 점진적으로 어려워지는 웨이브 시스템
- 골드 관리 및 타워 업그레이드 (향후 구현 예정)
- 생명이 다하면 게임 오버
- 배경 음악 지원

## 실행 방법

1. `index.html` 파일을 웹 브라우저에서 열어주세요.
2. 오른쪽의 타워 메뉴에서 타워를 선택합니다.
3. 게임 화면에서 클릭하여 타워를 배치합니다.
4. "다음 웨이브 시작" 버튼을 클릭하여 웨이브를 시작합니다.
5. 적이 끝에 도달하지 못하도록 막하면서 생명을 유지하세요.

## 배경 음악 추가 방법

배경 음악을 추가하려면:
1. MP3 형식의 배경 음악 파일을 준비하세요
2. 파일을 `assets/bgm.mp3` 로 저장하세요
3. 게임을 새로고침하면 배경 music이 자동으로 재생됩니다

참고: 브라우저 정책으로 인해 사용자 interaction(클릭, 키 입력 등)이 있을 때만 오디오가 재생됩니다. 게임을 시작할 때 화면을 클릭하면 배경 music이 시작됩니다.

## 파일 구조

- `index.html`: 메인 HTML 파일
- `css/style.css`: 스타일시트
- `js/entities.js`: 게임 엔티티 (타워, 적, 발사체 등)
- `js/waves.js`: 웨이브 관리 시스템
- `js/ui.js`: 사용자 인터페이스 처리
- `js/main.js`: 메인 게임 로직
- `assets/`: 오디오 파일 폴더 (bgm.mp3 위치)

## 클라우드플레어 배포를 위한 참고사항

이 게임은 정적 파일들로만 구성되어 있으므로 클라우드플레어 페이지나 정적 사이트 호스팅에 쉽게 배포할 수 있습니다.
모든 파일을 클라우드플레어에 업로드하면 즉시 플레이 가능합니다.
